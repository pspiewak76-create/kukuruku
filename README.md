# Symulator Łazika Marsa – Gigathon 2026 (III Etap)

Projekt na trzeci etap konkursu Gigathon 2026. Program to tekstowy symulator wyprawy kosmicznej, który działa w konsoli.

---

## Jak to włączyć

1. Otwórz folder z projektem w swoim programie do kodowania
2. Otwórz plik main.py.
3. Kliknij zielony przycisk Run albo ikonkę zielonej strzałki na górze ekranu. Gra od razu poprosi cię o podstawowe dane do działania gry następnie gra ruszy w konsoli na dole i można zacząć sterować łazikiem.
4. Wszystkie elementy sterowanie obsługiwania się program są tłumaczone w terminalu.
